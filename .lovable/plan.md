

# Lathika Reddy — Personal Portfolio Website

## Overview
A modern, responsive personal portfolio built as a single-page React app with dark/light theme toggle, smooth scroll, and subtle animations.

## Sections

### 1. Hero Section
- Name: **Lathika Reddy**, Title: **Full Stack Developer**
- Creative tagline
- Two CTA buttons: "View Projects" and "Contact Me" (smooth scroll to sections)
- Subtle fade-in animation on load

### 2. About Section
- Professional summary as a CS student passionate about web dev and problem solving
- Clean card layout with accent styling

### 3. Skills Section
- 6 skill cards in a responsive grid: HTML, CSS, JavaScript, React, Git, GitHub
- Icons for each skill, hover scale effects

### 4. Projects Section
- 3 project cards: Portfolio Website, To-Do App, Calculator App
- Each with title, description, and "View Code" / "Live Demo" buttons
- Hover lift effect on cards

### 5. Contact Section
- Email, LinkedIn, and GitHub links with icons
- Clean centered layout

### 6. Footer
- Name + copyright year, minimal design

## Design & UX
- **Dark/Light mode toggle** in a sticky navbar
- Professional color scheme with accent color
- Smooth scrolling between sections
- Fade-in animations on scroll using Intersection Observer
- Fully responsive: mobile-first grid layouts
- Hover effects and transitions throughout

## Technical Approach
- Built within the existing React + Tailwind stack (single Index page with component sections)
- Dark mode via Tailwind's `dark` class toggle
- Custom scroll animation hook
- All sections as separate components for clean structure
